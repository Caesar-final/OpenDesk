# OpenDesk

**Open-source Helpdesk & Ticketing System**  
Built with Spring Boot 3 + Angular 18 + PostgreSQL  
Single Docker image · 100% free · MIT licensed

A clean, modern, and ridiculously easy-to-deploy helpdesk for teams that don’t want to pay for Zendesk.
